# Cron Jobs

Projeto visa orquestrar a execução de scripts de acordo com um agendamento pré estabelecido.

## Iniciando Docker

Para iniciar um novo container com os cronjobs, basta executar os seguintes comandos.

```bash
docker build -t cronjobs .
docker run cronjobs
```
